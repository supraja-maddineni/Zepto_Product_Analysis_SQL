**Zepto Product Analysis - SQL Data Analysis Project**
**Project Overview**

This project performs an in-depth SQL-based data analysis on Zepto's product catalog. The analysis explores inventory patterns, pricing strategies, discount structures, and product availability across multiple categories to derive actionable business insights for quick-commerce optimization.

**Dataset**
The dataset (zepto_v2.csv) contains 2,215 product records with the following attributes:

Category: Product category (Fruits & Vegetables, Cooking Essentials, Dairy, Personal Care, etc.)

name: Product name

mrp: Maximum Retail Price (in paise, converted to rupees)

discountPercent: Discount percentage offered

availableQuantity: Current stock quantity

discountedSellingPrice: Final price after discount (in paise)

weightInGms: Product weight in grams

outOfStock: Boolean flag for stock availability

quantity: Pack/unit quantity

**Key Objectives**
Clean and prepare raw product data for analysis

Analyze inventory patterns and stock availability

Evaluate category-wise performance and revenue estimation

Identify discount trends and pricing strategies

Calculate product value metrics (price per gram)

Generate actionable business recommendations

**Technologies Used**
Database: PostgreSQL

Language: SQL

Tools: pgAdmin

Version Control: Git
